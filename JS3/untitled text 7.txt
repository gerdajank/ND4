
Duotas objektas:
var preke = {
title: "Nešiojamas kompiuteris",
description: "NVIDIA GeForce GTX 1070 grafinis adapteris su Max-Q4 technologija, Intel Core 8th Generation procesorius ir pažangi aušinimo sistema suteikia reikiamą galią, nesvarbu, kur būtumėte.",
discount: 30,
isItNew: true,
color: 0,
price: 699.00,
currency: "EUR"
};
Pagal duotą objektą, naudodami document.write turite sugeneruoti prekės aprašymą elektroninei parduotuvei. (pvz (tik kaip pvz) cia: https://i.stack.imgur.com/GiIsV.jpg )
Padarykite, kad spalva būtų pasirenkama ( <select> ).
color - savybės reikšmės (0 - "juoda", 1 - "raudona", 2 - "žalia", 3 - "sidabrinė");
isItNew - sąvybės reikšmė true, tai tekstas turi būti žalias, kitu atveju juodas.

Ši užduotis pagal sudėtingumą prilygsta ~11 JS užduočiai, tad jeigu nepavyksta šios užduoties išspręsti dabar pabandykite prie jos grįžti po kiekvienos paskaitos, kuomet žinosite JS programavimo vis daugiau ir daugiau.
